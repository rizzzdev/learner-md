# 1. Hello World: Langkah Pertama Kita

Modul ini adalah titik awal di mana kita memberikan perintah pertama kita kepada komputer.

## Analogi: Juru Masak dan Pelayan Restoran
Bayangkan komputer itu seperti sebuah dapur restoran yang canggih, tapi sangat kaku. Jika Anda (sang programmer) tidak memberikan perintah/pesan yang jelas, komputer tidak akan melakukan apa pun. Perintah `Hello World` ini ibarat Anda mengetes mikrofon ke arah dapur dan bilang: **"Tes tes, apakah koki bisa mendengarku?"** 

Di dunia JavaScript, kita menggunakan fitur bernama `console.log()` untuk mengetes hal tersebut. Komputer akan langsung merespon tulisan persis seperti yang kita suruh.

## Implementasi Kode
```javascript
// Meminta komputer untuk mencetak / menampilkan teks "Hello World!"
console.log("Hello World!");

// Kita juga bisa mencetak perintah lain
console.log("Hei komputer, saya sedang belajar JavaScript nih!");
```

### Penjelasan Singkat
- `console.log` : Artinya kita meminta komputer untuk mencatat (log) sesuatu ke dalam konsol (monitor/layar).
- `"Hello World!"` : Isi pesan yang mau kita sampaikan. Teks dalam JavaScript selalu diapit tanda kutip (`""` atau `''`).
- `;` (Titik koma) : Ibarat tanda titik di akhir kalimat. Menandakan bahwa satu instruksi sudah selesai.

Cobalah dan rasakan komputer menyapa Anda kembali!
